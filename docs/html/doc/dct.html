9. dct¶

Geoff Leach 的Delaunay triangulation c code。

考虑了2d情况最差情况并对其进行了优化，详见论文：Improving Worst-Case Optimal Delaunay Triangulation Algorithms 。在 GRASS GIS 中使用的即是此版本。
